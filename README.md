# ExtrFunReg 
This package performs extremile scalar-on-function regression from "Extremile scalar-on-function regression" by M.L. Battagliola and M. Bladt (2024) accepted for pubblication by the Canadian Journal of Statistics.

To download the package, type in R

```R
install.packages("devtools")
install_github("LauraBattagliola/ExtrFunReg")
```
